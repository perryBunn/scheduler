## Install

### Requirements

1. Python 3.10
	> See the `requirements.txt` file for the exact list of python dependencies needed for this project to run properly. 


### Setup

```bash
git clone git@github.com:perryBunn/scheduler.git
cd scheduler
python -m pip install -r requirements.txt
```
